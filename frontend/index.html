Targoo V2 Methodology

Targoo V2 is a high-performance ESG data refinery engine designed for CSRD/ESRS E1 compliance. It uses a hybrid approach to carbon accounting:

  • GHG Protocol: Full alignment with Scope 1, 2, and 3 accounting standards.
  • PCAF: Specialized financial sector attribution for Scope 3 Category 15.
  • EEIO & Hybrid Models: Intelligent fallback to environmentally-extended input-output models when primary data is unavailable.
  • WORM Ledger: Write-Once-Read-Many architecture for immutable audit trails.

All calculations are performed using the latest IPCC AR6 GWP100 factors and verified emission factor databases.
